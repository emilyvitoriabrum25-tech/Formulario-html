# Formulário de Cadastro

Este projeto foi desenvolvido durante o curso de Programador Web I da FAETEC.

## 💻 Tecnologias utilizadas
- HTML
- CSS

## 📌 Descrição
Formulário simples de cadastro contendo campos de entrada, seleção e validação básica de dados.

## 🎯 Objetivo
Praticar os conceitos iniciais de desenvolvimento web, como estrutura de formulários e estilização com CSS.

## 🚀 Status
Projeto finalizado para fins de estudo.
